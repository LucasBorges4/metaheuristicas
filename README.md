# metaheuristicas

trabalho 1, 2 e 3

1 é analisar uma função matemática complexa, utilizando os conceitos até a primera parte do curso.
2 é um problema competitivo e fica a seu critério utilizar alguma metaheuristica ou inovar.
3 é o mesmo problema, porém nesse caso decidi inovar kkk criei alguns conceitos para algoritmos genéticos. Como a ideia de microquimerismo, onde a femea absorve o genes do macho com um melhor score (hipergamia) ela pode ou não ter vários parceiros (geralmente a maioria dos casos é 1/2 e num maximo 3 para esse caso sendo muito dificil existe qualquer acima de 3 mas acontece) e ao longo das gerações, no caso a femea absorveria genes para ela e repassaria para a prole da geração seguinte, , um macho pode ter várias e para esse caso acho que testei eventualmente se os machos com menor score tem 0 parceiras e isso aconteceu ai tem vários parametros para criar maior caos, mas que na realidade é necessária muitas iterações para chegar num resultado recorrente. Faltou melhorar bastante coisa. Por fim minha conclusão é que esta metaheuristica ainda deve muito em termos de eficiencia para encontrar o melhor caminho para um ótimo resultado.
